The KSU Tournament Bot is a Discord bot that allows users to create and manage tournaments currently only for League of Legends. The bot is able to link user's Discord profile with their riot accounts, host tournaments, create teams for tournaments, and track player stats. It is designed to automatize certain aspects of the KSU League Discord's process of hosting tournaments. The bot makes starting tournaments and creating teams more efficient, making a smoother experience for both players and organizers

The bot is built using Python and utilizes various libraries to handle tasks such as data storage, user and Discord interaction, and tournament management. The bot uses aioSQLite and Google Sheets as a database to allow low barriar of entry to storing data.
